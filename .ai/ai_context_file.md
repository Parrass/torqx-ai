# Torqx MVP - Contexto para IA

## Projeto
SaaS para oficinas automotivas com IA nativa

## Stack Tecnológica
- Frontend: Vite + React + TypeScript + shadcn/ui + Tailwind
- Backend: Supabase (PostgreSQL + Auth + Edge Functions)
- Automação: n8n
- IA: OpenAI API
